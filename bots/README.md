# bots


Playground for bot fighting. Documentation is included in the code and generated by Doxygen

## rules

- Bots can move in 8 directions.
- Bots can attack and be attacked.
- Bots start with experience = 1
- Bots evolved through experience:
    - attack = ATTACK_MULTIPLIER * experience
    - defense = DEFENSE_MULTIPLIER * experience
- Bots recover energy when they kill a number of bots
- Bots gain experience for killing other bots
- The game is over when there is only one team in the field
